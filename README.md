# San Alejo App

Aplicación móvil desarrollada en React Native con Expo para gestionar contenedores (cajas, maletas, cajones) y los objetos almacenados dentro de ellos.

---

# 1. Clonar el proyecto desde GitHub

```bash
git clone https://github.com/WilmerGelves/AppSanAlejo-ReactNative
```
# 2. Requisitos previos
Antes de ejecutar el proyecto, asegúrate de tener instalado:

- Node.js (versión 18 o superior recomendada)
- npm 
- Aplicación Expo Go en tu celular (Android o iOS)

Verificar instalación:
```bash
node -v
npm -v
```
# 3. Instalar dependencias
```bash
npm install
```
# 4. Ejecutar la aplicación
Esto nos crea un código QR para escanerlo con la app de Expo GO.
```bash
npx expo start
```
# 5. Abrir la aplicación
  1. Abre el proyecto que clonaste en tu entorno de desarrollo preferido.
  2. Descarga la app Expo Go en tu celular.
  3. Ejecuta el mencionado 
  4. Escanea el código QR que aparece en la terminal o navegador 
  5. Escanea el código QR que aparece en la terminal o navegador
  6. La app se ejecutará automáticamente
